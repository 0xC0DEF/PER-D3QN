## How to use  
1. git clone https://github.com/0xC0DEF/PER-D3QN  
2. cd PER-D3QN  
3. open Snake.ipynb and run all cell (start training)  
4. open and run Test.ipynb to test learning agent (You can use Test.ipynb during training)
